PopSmartKids API Information

API Version 1.0

JSON response for product information

PopSmartKids offers a request to retrieve product information through our API. Making this request will return PopSmartKids product info, including title, description, and a link to its page on our website.

To make this request, please visit smart2.icoolshow.net/API/fullJson.php


Response format will appear as follows:

{
"id":1,
"title":"Product Name",
"description":"Product description",
"url":"product url"
}


Wordpress offers a full Rest API as well, with full product information less pertinent to most users.

To make the Wordpress JSON request, please visit smart2.icoolshow.net/products/wp-json/wp/v2/posts

*To retrieve individual product data, add a dash (/) and the product id number to the end of that URL (ex- smart2.icoolshow.net/products/wp-json/wp/v2/posts/1).
